# scala-sbt-alpine

A minimal sbt alpine docker image based on OracleJDK 8  and [frolvlad/alpine-scala](https://hub.docker.com/r/frolvlad/alpine-scala/)

current environment parameters (can be configured)

```dockerfile
SCALA_VERSION=2.13.1
SBT_VERSION=1.3.8
```
